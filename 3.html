Basic Math

In this lesson, we're going to be talking about performing basic math operations using Python.

Operators

Here is a quick rundown of most of the major operators in Python.

  • + : addition
  • - : subtraction
  • / : division
  • * : multiplication
  • % : modulus

Some useful terminology: These symbols are all called operators. The two numbers they go between are called operands. So in the example 2 + 3, + is the operator and 2 and 3 are the operands.

Modulus (%)

The % operator, pronounced as modulus, does not function as a percent operator. Instead, it is essentially a "remainder" operator that returns the remainder of a division. You will understand what this means after looking at some examples.

Examples:

  • 5 % 2 = 1

    5 divided by 2 is 2, with a remainder of 1,so 5 % 2 returns 1

  • 6 % 2 = 0

    6 / 2 = 3 with no remainder, because 6 is wholly divisible by 2. So there is no remainder, meaning 6 % 2 returns 0.

  • 4 % 10 = 4.

    4 goes into 10 zero times, with a remainder of 4. Think about third grade division for these and it makes it very easy.

What is Modulus useful for?

I think it is most used to determine whether a number is even or odd.

Given a number n, we can say that:

  • if n % 2 is 0, n is even.
  • else, if n % 2 is 1, n is odd.

Think about a few examples and this should make perfect sense.

  • 4 % 2 == 0, so 4 is even. We can say 4 is an even number because it divides evenly by 2 with no remainder.
  • 11 % 2 == 1, so 11 is odd. We can say 11 is an odd number because it does not divide evenly by 2.
