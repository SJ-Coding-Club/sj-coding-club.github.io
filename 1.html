Setting up Python

Check for current installation

Check if you already have Python installed. You can do this by opening a command line and typing

python --version

If you have anything under Python 3.8 or no Python at all, continue reading.

Installing Python

We are going to be installing the latest version of Python, Python 3.8. These instructions are currently for Windows, as that is the OS installed on school computers. I'll add Mac instructions if needed, but they aren't much different anyways.

Method 1

  1. Download the installer by scrolling to the bottom of the page and clicking 'Windows x86-64 executable installer'
  2. Launch and run the installer. Make sure you select the box that tells you to add Python 3 to your Path.

Method 2 (Windows 10 Only)

  1. Open the Microsoft Store app.
  2. Search for Python and select Python 3.8
  3. Install it.

Installing a Text Editor

Text editors make it easy for us to write and save our code. Two of my favorites are Visual Studio Code and Sublime Text. Pick one of them and download+install it. I know VSCode can be installed on school computers without an admin, but I am not quite sure about Sublime Text. If neither work, use Atom.
