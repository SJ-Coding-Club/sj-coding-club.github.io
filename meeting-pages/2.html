Creating a Python file

Create a new folder called Coding Club Python. Inside the folder, right click, hover over New, and select Text Document. Rename this file to 'hello.py' without the quotes. Now, you can open it with the text editor you installed by selecting the file, right clicking it, and then and selecting 'Open With' your text editor.

hello.py

  • Getting Output

Type (Not copy and paste) the following code into hello.py:

-----------------------------------------
print("Hello World")
print("This is a message.")
print('I can use single quotes too.')
print('I love printing text!')
-----------------------------------------

To run this code, open up a command line and change your current directory to the location of hello.py (It's really important to know how to do this with the command line, so I will show you how. You'll get the hang of it pretty quickly).

Once in the right directory, simply run the following command:

-----------------------------------------
python hello.py
-----------------------------------------

You should see this output:

-----------------------------------------
Hello World
This is a message.
I can use single quotes too.
I love printing text!
-----------------------------------------

Commenting Your Code

  • What do comments do?
  • Why use comments?

Create another file named comments.py and write the following code in it:

-----------------------------------------
# Comments can be used to explain what your code is doing in English.
# They help you understand your code when you come back to it later.
# They also help other people understand your code.

print('Comments can go next to code too.') # Comment

# Comments can also be used to temporarily disable code
print('This code is executed')
# print('This code is not executed')
-----------------------------------------

If you run this code in the same method as last time, using the command python comments.py the output should be:

-----------------------------------------
Comments can go next to code too.
This code is executed
-----------------------------------------
