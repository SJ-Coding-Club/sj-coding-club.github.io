Joining Our GitHub Organization

  1. Create an account on GitHub (Use your student email to get free GitHub Pro)
  2. Go to the repository for this website in the SJ Coding Club GitHub organization.
  3. Click Issues
  4. Click New issue
  5. Create an issue titled: Add Me and submit it.

    6. You'll then automatically receive an invitation to join the organization via email.
